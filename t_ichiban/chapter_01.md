## Chapter 1 企業活動

### lesson 01-01

| 用語                 | 説明                                                                         |
| -------------------- | :--------------------------------------------------------------------------- |
| 企業活動             | 商品やサービスを作り それを顧客に売ることでお金を稼ぐ会社の営み              |
| 株式会社             | 株主にお金を出してもらって事業を運営する会社                                 |
| 株式会社を構成する人 | 経営者、社員、株主                                                           |
| 経営理念             | 会社の運営方針を決定するための最も基本的かつ大切な指針であり、会社の存在意義 |
| ビジョン             | 会社が目指す将来の姿                                                         |
| 経営戦略             | ビジョンを実現するための具体的な行動計画                                     |

| SDGs                   | Sustainable Development Goals の略。持続可能な開発目標。<br>持続可能な世界を実現するための 17 の国際目標 |
| ステークホルダー       | 従業員、株主、取引先、消費者、行政機関、地域社会など企業と利害関係にある人                               |

### lesson 01-04

| 用語                         | 説明                                                                                                                                 |
| ---------------------------- | :----------------------------------------------------------------------------------------------------------------------------------- |
| 経営組織                     | 経営資源であるヒトの集まり                                                                                                           |
| 組織形態                     | 経営組織の形態のこと。職能別組織、事業部制組織、マトリックス組織、プロジェクト組織、持ち株会社がある。                               |
| 職能別組織                   | 業務を専門的な機能に分けて、各機能を単位として構成する組織。仕事の種類で仕事を分けて、社員の専門性を生かす。機能別組織とも呼ばれる。 |
| 職能別組織の<br>意思決定権   | 事業が 1 つしかないため、経営者が持つ                                                                                                |
| 事業部制組織                 | 事業部を単位として構成する組織。職能別組織の上に事業部の階層を増やすこともある。                                                     |
| 事業部制組織の<br>意思決定権 | 事業が複数あるため、事業部が持つ。また、事業部は利益責任を負わされる                                                                 |
| 事業                         | 複数の業務                                                                                                                           |
| 業務                         | 手作業やシステム(ハードウェアやソフトウェア)                                                                                         |
| マトリックス組織             | 社員が職能部門と事業部門の両方に所属する組織                                                                                         |
| プロジェクト組織             | ある問題を解決するために各部門から専門家を集めて結成する一時的な組織                                                                 |
| 持ち株会社                   | 子会社の株式を保有する親会社。企業の買収や合併がしやすくなる。                                                                       |
| CEO                          | Chief Executive Officer の略。経営者(社長)。業務の最終的な責任を負う役職                                                             |
| CIO                          | Chief Information Officer の略。情報システムの統括をする最高責任者                                                                   |

### lesson 01-05

| 用語           | 説明                                                                                                 |
| -------------- | :--------------------------------------------------------------------------------------------------- |
| パレート図     | 棒グラフと折れ線グラフを組み合わせたグラフ                                                           |
| ABC 分析       | データをパレート図で表し、データの重要度に応じて 3 つのグループにランク分けする分析手法              |
| ヒストグラム   | データをいくつかの区間に分けて、その区間ごとのデータの数を棒グラフで表したグラフ                     |
| レーダチャート | 複数のデータを蜘蛛の巣のような放射状に表した図。データ全体のバランスを見れる。                       |
| 散布図         | 2 種類のデータの関係性を表す図。相関関係を確認できる。                                               |
| 回帰直線       | 散布図内の点が多く集まっているところに引いた直線                                                     |
| 回帰分析       | 回帰直線を用いて過去のデータがどのような原因によって引き起こされているか分析し、将来を予測する手法。 |

### lesson 01-07

| 用語       | 説明                                                                                              |
| ---------- | :------------------------------------------------------------------------------------------------ |
| 売上       | 商品やサービスを販売して得たお金                                                                  |
| 費用       | 商品やサービスを売るために必要になるお金                                                          |
| 変動費     | 販売数に比例して増える費用                                                                        |
| 変動比率   | 変動費/売上。売上に占める変動費の割合                                                             |
| 固定費     | 販売数に関係なく必ず発生する費用                                                                  |
| 利益       | 売上から費用を差し引いた金額                                                                      |
| 損益分岐点 | 売上=費用。売上の額と費用の額が等しくなる売上高(販売個数)。グラフ上の売上の線と費用の線が交わる点 |

### lesson 01-08

| 用語                               | 説明                                                                                                                                              |
| ---------------------------------- | :------------------------------------------------------------------------------------------------------------------------------------------------ |
| 財務諸表                           | 経営成績を報告するための書類。貸借対照表、損益計算書、キャッシュフロー計算書、株主資本等変動計算書の 4 つ。                                       |
| 貸借対照表                         | Balance Sheet(B/S)。会社の全財産が記載された表。                                                                                                  |
| 損益計算書                         | 会社の利益が記載された書類。会社がどんな方法でお金を稼いだかを明らかにする。                                                                      |
| キャッシュフロー計算書             | 会社の現金の出し入れが記録された書類。会社の一定期間の 現金収支を明らかにする。                                                                   |
| 総資産                             | 会社が持っている全財産                                                                                                                            |
| 他人資本                           | 返済が必要なお金                                                                                                                                  |
| 自己資本                           | 返済する必要がないお金                                                                                                                            |
| 総資本                             | 他人資本と自己資本を合わせた資本                                                                                                                  |
| 資本金                             | 株主が会社に出資した金額。純資産                                                                                                                  |
| 自己資産比率                       | 総資本に対する自己資本の割合。自己資本比率(%) = (自己資本/総資本)\*100                                                                            |
| 売上高                             | 商品を売って得た金額                                                                                                                              |
| 売上原価                           | 商品を作るための費用。変動費。                                                                                                                    |
| 売上総利益                         | 売上高-売上原価。粗利益のこと。                                                                                                                   |
| 販売費及び一般管理費               | 商品を販売・管理するための費用(販管費)。固定費。                                                                                                  |
| 営業利益                           | 売上総利益-販管費。本業で稼いだ利益。                                                                                                             |
| 営業外収益                         | 本業以外で稼いだ収益                                                                                                                              |
| 営業外費用                         | 本業以外でかかった費用                                                                                                                            |
| 経常利益                           | 営業利益+営業外収益-営業外費用。本業と副業を合わせた会社全体の競争力。                                                                            |
| 特別利益                           | 臨時の利益                                                                                                                                        |
| 特別損失                           | 臨時の損失                                                                                                                                        |
| 税引前当期純利益                   | 税金を引く前の会社の利益                                                                                                                          |
| 法人税等                           | 税金を払うための費用                                                                                                                              |
| 当期純利益                         | 全ての利益から全ての費用を差し引いた金額                                                                                                          |
| 黒字倒産                           | 損益計算書では利益が出ているが倒産すること                                                                                                        |
| 営業活動による<br>キャッシュフロー | 本業に関わるお金の残高を計算                                                                                                                      |
| 投資活動による<br>キャッシュフロー | 設備投資や株、債券の売買後の残高を計算                                                                                                            |
| 財務活動による<br>キャッシュフロー | 資金調達に関わるお金の残高を計算                                                                                                                  |
| ROE                                | Return On Equity の略。自己資本比率。会社が株主から集めたお金でどれだけの利益を上げたのかを表す指標。 <br>ROE(%) = (当期純利益 / 自己資本) \* 100 |
